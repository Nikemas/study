Understanding Alignment

When working with Flexbox, one of the most common points of confusion is the difference between justify-content and align-items. These two properties control how items are distributed within a flex container.

The Main Axis vs. Cross Axis

Flexbox operates on two axes. The Main Axis is defined by the flex-direction property (row by default). The Cross Axis runs perpendicular to it.

  • justify-content aligns items along the Main Axis.
  • align-items aligns items along the Cross Axis.
styles.css 
.flex-container {
  display: flex;
  height: 400px;
  
  justify-content: center;
  
  align-items: center;
}

In the example above, because the default direction is row, justify-content: center moves the items to the middle of the horizontal line, while align-items: center moves them to the middle of the vertical space.
