How do I center a div using CSS Flexbox?

To center a div using Flexbox, you need to apply three properties to the parent container:

  • display: flex;
  • justify-content: center; (horizontal)
  • align-items: center; (vertical)
style.css 
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh; 
}
